# Contribuir a NEO

¡Gracias por tu interés! Cualquier mejora es bienvenida.

## Cómo contribuir

1. Haz un fork del repo
2. Crea una rama: `git checkout -b feature/mi-mejora`
3. Haz tus cambios y commitea: `git commit -m "feat: descripción corta"`
4. Sube la rama: `git push origin feature/mi-mejora`
5. Abre un Pull Request

## Ideas de contribución

- Nuevas herramientas en `tools/tools.py`
- Soporte para más proveedores LLM
- Mejoras en el sistema de memoria
- Traducción de los ficheros de config a otros idiomas
- Tests

## Convenciones de commits

Usa prefijos estándar: `feat:`, `fix:`, `docs:`, `refactor:`, `chore:`
